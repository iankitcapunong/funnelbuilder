# Easy Funnel Builder — AI Funnel Prompt Builder

A single-file web app that helps you design a high-converting sales funnel and
either **generate a live, fully-styled funnel preview** or **copy a ready-to-paste
prompt** for any AI builder (ChatGPT, Claude, Lovable, v0, Bolt, …).

No build step, no dependencies — just open [`index.html`](index.html) in a browser.

## Features

- **Describe your funnel** — pick a business niche or type your own.
- **Choose a funnel type** — Lead Gen, Sales Page, VSL, Webinar, Launch, Free Trial,
  Book a Call, or E-com Product. Each loads a strategy-appropriate set of steps.
- **Pick a style** — six visual styles (Clean, Bold, Warm, Playful, Luxury, Vibrant).
- **Build your funnel** — add and drag-reorder conversion blocks (Hero Offer, Problem,
  Social Proof, Benefits, How It Works, Offer Stack, Testimonials, Pricing, Guarantee,
  Urgency, FAQ, Lead Capture, Final CTA, Footer).
- **Generate funnel** — renders a real, responsive funnel page in a full-screen preview.
  - **Unique every time** — copy, layout, typography, shapes, and visual rhythm are
    randomized on each generation.
  - **Shuffle** — re-roll a fresh variation instantly.
  - **Desktop / Mobile** preview toggle.
  - **Download HTML** — export the funnel as a standalone, self-contained file.
- **Copy prompt** — get a structured, conversion-focused prompt for any AI builder.

## Theme

Eye-friendly **green · gold · white** palette, with green and gold accents blended
across the builder sections.

## Usage

```
Open index.html in any modern browser.
```

That's it — everything runs client-side.
